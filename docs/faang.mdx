# FAANG

[FAANG](https://www.investopedia.com/terms/f/faang-stocks.asp) is an acronym that refers to the stocks of five technology companies: Meta (FB), Amazon (AMZN), Apple (AAPL), Netflix (NFLX), and Alphabet (GOOGL).

The charts below show historical share price and revenue data for each of the FAANG companies. The charts were created in [JupyterLab](https://jupyter.org/) using [pandas](https://pypi.org/project/pandas/) and [plotly](https://plotly.com/graphing-libraries/) python graphing library and exported to PNG.

Historical share price data was collected from Yahoo! Finance APIs using [yfinance](https://pypi.org/project/yfinance/). Historical revenue data was scraped from [Macrotrends](https://www.macrotrends.net/) using [requests](https://pypi.org/project/requests/) and [beautifulsoup4](https://pypi.org/project/beautifulsoup4/).

The Jupyter Notebook used to create these charts can be found on [Gist](https://gist.github.com/wemhr30/ffc59b1ff850ffb7a2fd188d4e696293).
