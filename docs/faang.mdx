# FAANG

[FAANG](https://www.investopedia.com/terms/f/faang-stocks.asp) is an acronym that refers to the stocks of five technology companies: Meta (FB), Amazon (AMZN), Apple (AAPL), Netflix (NFLX), and Alphabet (GOOGL). The charts below show historical share price and quarterly revenue data for each of the FAANG companies. 

I created the charts in [JupyterLab](https://jupyter.org/) using [pandas](https://pypi.org/project/pandas/) and [plotly](https://plotly.com/graphing-libraries/) python graphing library and exported them to PNG.

I collected the historical share price data from Yahoo! Finance using [yfinance](https://pypi.org/project/yfinance/) and scraped the historical revenue data from [Macrotrends](https://www.macrotrends.net/) using [requests](https://pypi.org/project/requests/) and [beautifulsoup4](https://pypi.org/project/beautifulsoup4/).

You can find the Jupyter Notebook I used to create these charts on [Gist](https://gist.github.com/wemhr30/ffc59b1ff850ffb7a2fd188d4e696293).
